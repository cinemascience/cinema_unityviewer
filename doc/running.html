Running the Viewer

Run the appropriate file for your OS

  • For Mac, run CinemaUnityViewer.app. The .app file is standalone. No other files are needed to run it
  • For Windows, run CinemaUnityViewer.exe (You will also need the data directory to go with it).

Loading a Database

Open the meta database file you wish to view by either entering the path into the text field or using the browse button.
If you wish to view a single database, check the Single Database box and open the database's info.json file
For more information on creating a meta database file, please see the Meta Database Specification guide. If you just want to test the viewer, try opening the sample data.

Viewing a Database

Use the WASD keys to move horizontally, and the QE keys to move vertically. Use the mouse to look around.
Left-click to open the menus to adjust variables and view information about each database. Right-click to close them again.
Check the Orbit This box on a database to lock the controls into an orbit around that database. Use WASD to orbit and Q/E to zoom in or out.

Saving and Loading Camera Positions

To save camera angles and positions to view later, first press the Camera Options button on the main menu and enter the name of the file you would like the positions to be saved to in the box. The file must end in '.ca'. While viewing the database, any of the F1 to F10 keys to record the current camera position and angle to the corresponding slot. Press Enter to write all recorded positions to the file.
To load and view previously saved camera angles, open the Camera Options menu again and enter the path of the save file into the text field or use the browse button. If you would like the camera to animate when moving to saved positions, use the slider to adjust how long you want the animation to take. When viewing the database, press any of the 1 to 10 keys to move the camera to the position saved in the corresponding slot.
